# Roadmap

1. **Terminal base:** estructura, backup, enlaces y healthcheck.
2. **Bash modular:** extraer de forma revisada aliases, prompt y herramientas.
3. **Robotics Context Engine:** detectar el contexto y exponer una fuente de verdad.
4. **ROS 2 helpers:** flujos de build, launch, topics y workspaces.
5. **Status bar inteligente:** mostrar contexto útil sin ruido.
6. **Layouts automáticos:** preparar paneles para build, launch, topics y logs.
7. **SSH, Docker y Git avanzado:** integrar robots remotos y flujos de desarrollo.
